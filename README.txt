Данный проект был создан в рамках курса по веб-разработке.
Задание было такое:
Задание 2. Создайте свой собственный веб-сайт, основанный на фреймах.
Сайт должен включать отдельные страницы – Хедера, Футера, Меню и Контента. В качестве контента должны выступать минимум 3 страницы с описанием различной информации.
Сайт должен обладать единой тематикой (сайт автолюбителей, сайт программиста, сайт портфолио и т. д.). 
Все страницы должны быть оформлена с применением нативных (родными средствами тегами, атрибутами FONT, bgcolor, background и т.д.) средств HTML.
